FAQ

Why does iOS direct install only work on certain browsers?

Direct install is meant for iOS. Safari is usually the most reliable option for opening the install link.

I installed it, but it won’t open on iOS. What now?

If iOS blocks the app, you may need to trust the developer profile in your device settings. If you’re stuck, join the Discord and we’ll help.

Android says “App not installed” or blocks it.

Make sure “Install unknown apps” is enabled for your browser or file manager, then try again. Also confirm you downloaded the correct build for your device.
