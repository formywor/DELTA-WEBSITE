Which option should I use?
  • IPA: use ESign, KSign, or any signer app (signer setup is not our department).
  • Direct Install: easiest install — after install you may need to trust the app in settings.
  • If one direct link fails, use the other.
  • After updating, check Updates.
How to install the IPA (Sideload)

The IPA method requires a signer app like ESign or KSign (or any signer app you use). Signer app setup and certificates are not our department.

Step 1

Tap Download iOS IPA and save the file.

Step 2

Open your signer app and import the IPA.

Step 3

Install from the signer app. If you have issues, use Support.

How to install with Direct Install

Tap Direct Install 1 or Direct Install 2 and follow the iOS install prompts. After installing, you may need to trust the app.

Step 1

Tap Direct Install 1. If it fails, try Direct Install 2.

Step 2

Accept the install popup and wait for the icon to appear.

Step 3

Go trust the app (steps below), then open Delta.

How to Trust the App (Direct Install)

Go to Settings → General → VPN & Device Management, then find the profile and tap Trust. After trusting, reopen the app.
