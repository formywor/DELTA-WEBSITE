How to install (general guide)

iOS installs can vary depending on your setup. The safest approach is to use the option you trust and follow the prompts carefully. If you’re new, use the Status page to confirm availability.

Step 1

Download the correct iOS file from the buttons above.

Step 2

Follow your install method prompts (allow permissions if prompted).

Step 3

If it fails, check Status + Updates, then try again later.

Troubleshooting
Install option not working right now

If an install option is temporarily unavailable, check iOS Status and Updates. Try again later instead of downloading repeatedly.

Download is slow or fails

Try another browser, switch between Wi-Fi and mobile data, and avoid multiple parallel downloads. If it still fails, check Support.

App installs but won’t open

Restart your device and check Updates for any known notes. If the issue continues, use Support.
