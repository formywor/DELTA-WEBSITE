Troubleshooting + FAQ
Android blocks the install

Android may ask you to allow installs from your browser (or file manager). Follow the prompt and enable it temporarily. If it still fails, see Support.

Install fails or says “App not installed”

Re-download the file, check you have enough storage, and try again. If you have multiple APK options, try the standard build first.

Which APK should I choose?

Use the standard APK unless your device specifically needs ARM32 or a region build. If you’re unsure, start with the standard one.
