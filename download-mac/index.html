Setup checklist
  • Download the macOS build
  • Open it and move the app to Applications (if prompted)
  • If macOS blocks it, allow it in Privacy & Security
  • Restart the app after allowing permissions
  • If problems continue, check Support
If macOS blocks the app
Step 1

Open System Settings → Privacy & Security.

Step 2

Find the blocked app warning and choose Allow / Open Anyway.

Step 3

Re-open Delta. If needed, use Support.

The download opens but the app won’t run

Check Privacy & Security and allow the app, then restart it. If it still fails, read Updates.
