Setup checklist
  • Download the macOS build
  • Open it and drag the app to Applications (if prompted)
  • If macOS blocks it, allow it in Privacy & Security
  • Restart the app after allowing permissions
  • Check Support if issues continue
How to open if macOS blocks it
Step 1

Open System Settings → Privacy & Security.

Step 2

Find the blocked app message and choose Allow / Open Anyway.

Step 3

Re-open Delta. If it still fails, use Support.
